# OpenCVFaceDetection

This program recognizes faces and saves the recording in avi format.

What you need:
  1. Camera (usb preferred)
 
 You just need to download the file, you can do it manuelly or with git clone https://github.com/DarkNoise-dev/OpenCVFaceDetection.git

![image](https://user-images.githubusercontent.com/42913703/85721437-a59b1880-b6f1-11ea-99ca-610067d80f83.png)

![image](https://user-images.githubusercontent.com/42913703/85721513-b64b8e80-b6f1-11ea-954d-e7cc2bcf4e13.png)

Now you need to donwload cv2. If you want to download it with pip type pip install opencv-python

After you done that you just need to open main.py
Go to line 19 and set cameraResolution to your camera resolution, it must be in a tuple.

![image](https://user-images.githubusercontent.com/42913703/85721831-01fe3800-b6f2-11ea-9868-f092a025160b.png)

Now you can execute the script, a window will popup. You can close the program by pressing q.

After you have closed the program, a file with the name output.avi appears in the records folder.

![image](https://user-images.githubusercontent.com/42913703/85722726-ddef2680-b6f2-11ea-832e-0df4ff9006ba.png)
